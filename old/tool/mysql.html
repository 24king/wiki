    OS系统管理员

    当前我有一个mysql,但是对于这个版本我没有任何映像了,所以希望可以彻底从系统清除他.

    清理 yum 

    >rpm -q mysql | no package found
>[root@limix mysql]# rpm -i MySQL-server-advanced-5.6.14-1.el6.x86_64.rpm 
package MySQL-server-advanced-5.6.14-1.el6.x86_64 is already installed

>[root@limix mysql]# rpm -U MySQL-server-advanced-5.6.14-1.el6.x86_64.rpm 
package MySQL-server-advanced-5.6.14-1.el6.x86_64 is already installed

>[root@limix mysql]# rpm -e MySQL-server-advanced-5.6.14-1.el6.x86_64.rpm 
error: package MySQL-server-advanced-5.6.14-1.el6.x86_64.rpm is not installed

>[root@limix mysql]# rpm -qa | grep -i mysql
MySQL-server-advanced-5.6.14-1.el6.x86_64
MySQL-client-advanced-5.6.14-1.el6.x86_64

    现在看看这个工具的设计,一定不要添加.rpm 否则查询不到 

    
>[root@limix mysql]# rpm -evv MySQL-server-advanced-5.6.14-1.el6.x86_64 --nodeps
...
>[root@limix mysql]# rpm -evv MySQL-client-advanced-5.6.14-1.el6.x86_64 --nodeps
...
>[root@limix mysql]# rpm -qa | grep -i mysql                                  
>[root@limix mysql]# rpm -ee MySQL-server-advanced-5.6.14-1.el6.x86_64 
...
>[root@limix mysql]# rpm -ee MySQL-client-advanced-5.6.14-1.el6.x86_64
...
这样就给清理干净了吧,招架不住了

    安装mysql | 制定目录安装好了 

    >[root@limix mysql]# rpm -ivh --prefix /opt/mysql MySQL-server-advanced-5.6.14-1.el6.x86_64.rpm MySQL-client-advanced-5.6.14-1.el6.x86_64.rpm 
error: package MySQL-server-advanced is not relocatable
