    系统管理员

    1. yum install svn -- 安装SVN服务器软件
    2. mkdir -p /svn -- 在系统的某一个目录建立SVN的根目录;
    3. useradd -m -s /bin/bash svnadmin -d /opt/svn -- 创建一个管理用户;
    4. chown -R svnadmin /opt/svn -- 将这个目录给管理员用户管理；
    5. svnserve -d -r /opt/svn/ --log-file run.log --d deamon -- r 指定根目录是/svn
    6. ps -ef | grep svnserve -- 检查SVN服务器是否在运行;
    7. killall svnserve
    8. 配置系统启动后自动运行SVN服务器。

    配置管理员

    1. 定义版本库实例 -- 建一个版本库.用于存放你的工程存放.
      1. cd /opt/svn
      2. svnadmin create repo -- 建一个版本库用于定义存放位置;
      3. cd test
    2. 配置版本库实例
      1. cd conf authz passwd svnserve.conf
      2. vim svnserve.conf 鉴权和访问控制.
        1. anno-access = read 非授权用户不能够访问.
      3. auth-access = write 授权用户可以写操作.
      4. password-db = passwd 指明密码文件所在的目录.
      5. authz-db = authz 访问控制文件.
      6. vim passwd 鉴权
      7. vim authz 访问控制

    开发人员

    1. 定义自己的工程目录.
      1. cd ~/project
      2. mkdir -p svntest/{trunk,branches,tags} -- 定义自己的工程文件
    1. 将工程导入到配置库;
      1. cd media/
      2. svn import vimwiki svn://localhost/repo/vimwiki --message "Vimwiki Project"
    1. 删除工程
      1. rm -rf svntest
      2. svn co
      3. svn move
      4. svn status
      5. svn update
      6. svn commit
